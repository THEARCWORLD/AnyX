# Minecraft ANY-X
一个Minecraft轻量优化整合包
### 介绍
> 一个基于多个加载器的轻量级优化整合包，专注提供一个通用的性能解决方案；建议用电脑来游玩，以获得最佳体验（光影与材质包）
### 测试环境
- 测试设备：一加Ace2Pro
- 内存：2048MB
- 处理器：SnapDragon8Gen2
- 渲染器：Mobile Glues/LTW改版
- 分辨率：1163*572
